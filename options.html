🖥️ Local LLM (Ollama Integration)

✨ Run AI Locally - No Limits, No Costs!
Ollama lets you run powerful language models on your computer. Once set up, enjoy:
  • 🚀 Unlimited requests - No rate limits
  • 💰 100% free forever - No API costs
  • 🔒 Complete privacy - Data stays on your device
  • ⚡ Fast responses - 0.5-2 seconds typically
  • 🌐 Works offline - No internet required
📥 Setup Instructions (One-Time, ~5 minutes)
Step 1: Install Ollama
Download from: https://ollama.com/download
Available for Windows, macOS, Linux
Step 2: Download a Model
Open terminal/command prompt and run:
ollama pull llama3.2:3b This downloads ~2GB. First time only!
Step 3: Test It
In terminal, run:
ollama run llama3.2:3b "Hello!" Should respond in ~1 second
Step 4: Enable Above ☝️
Check "Use Local Ollama" and save settings
💡 Tip: Ollama runs in the background. You can check status with:
ollama list (shows downloaded models)
ollama ps (shows running models)
