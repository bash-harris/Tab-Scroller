# Tab Scroller: Smart Browser Assistant

A Manifest V3 browser extension that lets users control their browser tabs using plain English commands, alongside intelligent tab session management.

## Features

- **Natural Language Tab Control:** Built a Manifest V3 Chrome extension using JavaScript and CSS that lets users control their browser tabs using plain English commands — grouping, sorting, snoozing, and cleaning up with a single sentence.
- **Smart Search & Retrieval:** Added a smart search feature that finds answers hidden across dozens of open tabs instantly, powered by a Python backend using PyTorch, TensorFlow, and sklearn.
- **Session Memory & Visual Timeline:** Built a session memory system that saves research sessions and lets users resume browsing exactly where they stopped; used OpenCV, scikit-image, and a C/C++ module to generate a visual page-by-page timeline of past research.

## Installation for Development

1. Clone or download this repository.
2. Open Chrome (or any Chromium-based browser) and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click on **Load unpacked** and select the folder containing these files.

## Technical Stack
- **Frontend/Extension:** JavaScript, CSS, HTML, Chrome Extension API (Manifest V3)
- **Backend/AI (Planned/Integrated):** Python, PyTorch, OpenCV, C/C++, scikit-learn
